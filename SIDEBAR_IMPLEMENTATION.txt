🚀 USAGE GUIDE
───────────────────────────────────────────────────────────────

For End Users:
  1. Klik hamburger menu (☰) di mobile untuk buka sidebar
  2. Klik menu item untuk navigasi ke fitur
  3. Sidebar otomatis tutup setelah klik di mobile
  4. Klik overlay untuk tutup sidebar
  5. Di desktop, sidebar selalu visible

For Developers:
  1. Edit resources/views/layouts/app.blade.php untuk modify sidebar
  2. Tambah menu item dengan format:
     <a href="{{ route() }}" class="...">
       <i class="fas fa-icon"></i>
       <span>Label</span>
     </a>
  3. Gunakan request()->routeIs() untuk active state
  4. Conditional display dengan @if(in_array()) untuk role
